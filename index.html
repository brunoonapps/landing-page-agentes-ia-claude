Antes que você pergunte.

As perguntas que recebo desde a primeira pessoa que avisei sobre o projeto.

Eu nunca usei a API da Anthropic. Vou conseguir acompanhar?
Se você sabe HTTP e JSON, sim. O Capítulo 3 cobre desde criar a conta Anthropic até o "hello agent" rodando. A partir daí o livro escala de forma progressiva.
O conteúdo é em vídeo?
Não. O livro é texto + código, propositalmente. Texto é mais denso, mais fácil de revisitar e respeita o tempo do leitor. Curso em vídeo está no roadmap, mas como produto separado.
Posso clonar e usar o código em projetos comerciais?
Sim. Os exemplos vêm com licença MIT. Use, modifique, venda — desde que não revenda o ebook em si.
O livro vai ficar desatualizado rápido?
O ecossistema muda, mas os fundamentos não. O livro foi escrito para que 80% do conteúdo continue válido em 18 meses. As atualizações pontuais (mudanças de API, novos releases) são distribuídas gratuitamente para compradores.
Funciona pra Python ou só TypeScript?
Os capítulos cobrem ambas as linguagens. A maior parte dos exemplos está em TS por ser o ambiente mais comum em produção, mas há equivalentes em Python no repositório.
